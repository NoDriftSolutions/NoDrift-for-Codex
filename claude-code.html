Demonstrator

The free starting point is meant to show core NoDrift behavioral control: source-aware answers, bounded permission, and a visible stop before public or external action.

Basic Upgrade

Basic is the first purchase package path after Demonstrator. It is framed as the next step for an individual Claude Code user who wants fuller project setup, continuity, approval, and verification support.

Extended

Extended is the higher individual tier for larger or higher-risk work, with stronger carryover, evidence, readiness, and claim-safety controls.

Teams / Enterprise

Teams / Enterprise is the organizational tier direction for shared work, role-aware approvals, escalation, oversight, and safer release or client-facing decisions.
