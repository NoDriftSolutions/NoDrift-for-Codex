Fit

Customization

Customization changes how NoDrift communicates and leads the work. It does not change the safety floor that keeps source truth, approvals, corrections, privacy, and evidence intact.

Adjustable Working Style

The user can tune answer length, question frequency, progress-update detail, tone, explanation level, leadership style, and how visible evidence labels should be during the work.

Working Profiles

A working profile is a starting balance. One project may need fast, quiet work. Another may need deeper support, more explanation, and stronger visible checkpointing. The profile helps the AI workspace begin with the right behavior instead of guessing.

Token Discipline

Customization can reduce wasted tokens by making communication shorter and more direct when the user wants speed. It should not reduce tokens by skipping startup checks, source verification, correction protocol, or approval gates.

Protected Floor

The protected floor remains active across profiles: local source truth controls over memory, broad claims need evidence, external actions require exact approval, private material stays private, and errors are reported through the correction protocol.

When To Adjust

Adjustment is useful when the AI is over-explaining, under-explaining, asking too many questions, moving too quickly, reporting too little evidence, or failing to lead with a clear next directive.
