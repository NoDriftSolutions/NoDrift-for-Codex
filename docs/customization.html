Fit

Customization

NoDrift can adjust how the active AI project app communicates, leads work, asks questions, explains uncertainty, and reports evidence without weakening the safeguards.

What The User Can Tune

The adjustable range includes working profile, work leadership, question handling, communication detail, language level, progress updates, explanation level, and evidence-status visibility.

What Changes Token Use Most

Communication detail has the largest effect on token use. Shorter answers, lighter technical language, briefer updates, and lower explanation levels can reduce cost, but token savings should come from communication choices rather than weaker verification or weaker safeguards.

What Does Not Change

Customization does not override source discipline, privacy, approval gates, verification, evidence requirements for broad claims, or the rule against inventing facts. It changes style and workflow support, not the protective floor.

Profile Logic

The working profiles range from Fast Work through Deep Support. They are meant to give the user a strong starting point, with manual changes available later if a project needs a different balance of speed, context, explanation, or visible evidence labeling.
