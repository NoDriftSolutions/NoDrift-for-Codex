How To Handle Source Conflicts

When two sources disagree, the safest move is not to guess. The safest move is to stop and name the conflict.

NoDrift conceptSource conflict check
User problemThe AI agent chooses one source without showing the conflict.
Reading depthPractical

The problem

Projects accumulate old notes, revised drafts, current files, and remembered instructions. If those sources disagree, the AI agent can accidentally combine them into a confident but false answer.

What should happen

A source conflict should be visible. The AI agent should name the conflict, identify the competing records, and ask which source controls before using either one as project truth.

How NoDrift helps

NoDrift treats conflict as a control event, not an inconvenience. It keeps the project from moving forward on blended or guessed authority.

Practical hint

Use this sentence: "Source conflict. Do not resolve by guessing. Show the competing sources and ask which controls."
