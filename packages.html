Package levels

Choose how much control the project needs.

The levels are meant to be easy to understand: try it, use it, deepen it, then scale it across a team.

Demonstrator

The free starting point. It shows the core NoDrift behavior: less drift, clearer approval, local project context, and a small continuity trail.

Basic

The first purchase package. It adds stronger source, approval, memory, correction, and verification controls for real individual project work.

Extended

The advanced individual package. It adds deeper continuity, behavioral Heartbeats, evidence tracking, readiness review, and control-dashboard support.

Teams / Enterprise

The organizational package. It adds role-aware approvals, team routing, manager visibility, shared memory, and stronger multi-person governance.
